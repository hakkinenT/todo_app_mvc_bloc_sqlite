# Flutter MVC Todo App with Bloc and Sqlite

## Project Status
In completion phase

## Features

This Project contains the following features:
- Register User
- Login
- Logout
- Create Todo
- Update Todo
- Delete Todo
- Delete All Todo
- Change Theme Light/Dark
